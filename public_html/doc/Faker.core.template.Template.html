Class: Template

Template

new Template()

Template class, used to format all API responses using an specific layout. The following keywords can be used in the JSON template as property values and they will then be replaced by their values: - {data} : the data returned after processing the request. - {info} : in case something went wrong, a message containing information about what happened. - {success} : a boolean indicating whether processing the request went fine or not. - {total} : the number of items returned.
